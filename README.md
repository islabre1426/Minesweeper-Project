# Minesweeper Project
A Minesweeper clone written using Web technologies.

This project serves as the Final Project for Algorithms & Data Structures course at International University - Vietnam National University, Ho Chi Minh City.

A live version of this project is available on https://islabre1426.github.io/Minesweeper-Project/

## Features
- Retain original Minesweeper features:
    - Three difficulties: Beginner, Intermediate, Expert
    - Flagging tiles suspected as mine
    - Counting time since first opened tiles

## Technologies
- Frontend: HTML + CSS + JS
- Build Tool: Vite

## Structure
- /docs: Documentation (Report, Slides, etc.)
- /src: Source code

## Run the project locally
1. Clone this repo:
```bash
git clone https://github.com/islabre1426/Minesweeper-Clone
```

2. Inside the repo, install dependencies via:
```bash
npm install
```

This assumes you have NodeJS and npm installed.

3. After installing dependencies, run:
```bash
npm run serve
```
